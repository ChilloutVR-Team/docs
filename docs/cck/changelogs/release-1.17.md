# Content Creation Kit 1.17

!!! note
    This page was adapted from a snapshot from archive.org and may be incomplete. [View on archive.org.](https://web.archive.org/web/20210725221841/https://docs.abinteractive.net/cck-changelogs/content-creation-kit-1.17)

These are the release notes for ABI CCK 1.17 RELEASE Build number 49.

## Introduction

This version of our CCK contains new components and component changes.

!!! warning
    With the rollout of 1.17 all previous versions of our content creation kit are incompatible. You are REQUIRED to update if you want to upload content at this point. This also affects UPDATES of content already uploaded to our cloud.

### New features

- Added Layer CVRPickup
- Added Layer CVRInteractable
- Added Layer PostProcessing
- Added Component & Prefab for CVR Portal Marker
- Added Component & Prefab for CVR Chair
- Added Component CVR GI Material Updater
- Added Component CVR World BGM Marker
- Added Interactable Action DisplayWorldDetailPage
- Added Interactable Action DisplayInstanceDetailPage
- Added Interactable Action DisplayAvatarDetailPage
- Added Interactable Action SitAtPosition

### Changes

- The behaviour of voice position and view position identifier in CVR Avatar has been adjusted
- Changed uploader routine to include stereoscopic panosphere images for portals
- Performance improvements
- Fixed world audio adjustment
