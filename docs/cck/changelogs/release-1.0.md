# CCK 1.0 Release Notes (possibly)

!!! note
    This page was salvaged from a Discord announcement channel and may be incomplete. Irrelevant parts have been removed.

Hello everyone!

With this post we are announcing the launch of the first version of the ChilloutVR Content Creation Kit as well as the launch of the new Community Hub 2.0!

Please report any errors or problems you may encounter.

Whats in the CCK now:

- Build tools for worlds and avatars
- Trigger object for interactions: CVRInteractable
- Mirror: CVRMirror
- Content descriptors: CVRAvatar & CVRWorld
- Object sync to sync pos, rot, scale over network: CVRObjectSync

A documentation for all current and upcoming features will come up in the next weeks.

See you all at the next test!
Have fun creating wonderful stuff!
