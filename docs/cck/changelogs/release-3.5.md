# CCK 3.5 Release Notes

Hello There!

We have released CCK 3.5 to our documentation page.

This CCK will allow you to upload content using Unity 2021. We recommend Unity 2021.3.23f1. Even tho you will be able to upload using Unity 2019 we highly recommend upgrading to Unity 2021.

Additionally, this CCK contains the following changes:

- Added the following Keys to Interactable on InputUp/Down actions:
    - LeftShift
    - RightShift
    - LeftControl
    - RightControl
    - Space
- Added SpawnablePickupMarker Component
- Added Suggestive and Nudity to avatar advanced tagging
- Added AnimatorDriver StateMachineBehavior
- Added DisplaySpawnableDetailPage to interactable actions
- Added snapping points to possible attachment targets
- Added PlayerMaterialParser Component
- Added Avatar viseme smoothness setting

The corresponding game Update for these new additions will follow shortly.
If you want to check things out early, feel free to use our experimental branch.
